# Champion Fitness Gym App

## 📱 О приложении

**Champion Fitness Gym App** - современное мобильное приложение для фитнес-клуба с полным функционалом управления членством, тренировками, статистикой и взаимодействием с клиентами.

## 🚀 Основные функции

### 👤 Управление пользователями
- Регистрация и аутентификация
- Профиль пользователя с настройками
- Система ролей и прав доступа

### 🎫 Система абонементов
- Информация о текущем абонементе
- Покупка и продление абонементов
- История абонементов
- Специальные предложения и акции

### 💪 Тренировки и занятия
- Каталог групповых занятий (йога, пилатес, HIIT и др.)
- Запись на занятия и мероприятия
- Каталог тренеров с возможностью записи
- Персональные тренировки

### 📊 Аналитика и статистика
- Статистика посещений (неделя/месяц/год)
- Средняя продолжительность тренировок
- Лучшее время посещения
- Реферальная система с XP очками

### 🤖 ИИ Ассистент
- Чат-интерфейс для получения советов
- Персональные рекомендации по питанию и тренировкам
- Адаптация под цели пользователя

### 🏢 Управление клубами
- Каталог всех клубов сети
- Информация об услугах и оборудовании
- Расположение и контакты

### 🔔 Система уведомлений
- Push, email и SMS уведомления
- Настройки времени и типов уведомлений
- Напоминания о занятиях и акциях

## 🔗 API Integration

Приложение требует интеграции с CRM системой через REST API. Все необходимые endpoints и их спецификации описаны в документации.

### Основные API категории:
- **Аутентификация** (`/api/auth/*`)
- **Пользователи** (`/api/user/*`)
- **Абонементы** (`/api/memberships/*`)
- **Тренировки** (`/api/classes/*`, `/api/events/*`)
- **Клубы** (`/api/clubs/*`)
- **Тренеры** (`/api/trainers/*`)
- **Уведомления** (`/api/notifications/*`)
- **Аналитика** (`/api/analytics/*`)

## 🚀 Быстрый старт

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/alnnui/champion_gym.git
   cd champion_gym
   ```

2. **Установка зависимостей**
   ```bash
   flutter pub get
   ```

3. **Настройка конфигурации**
   - Обновите `lib/main.dart` с вашими API endpoints
   - Настройте переменные окружения

4. **Запуск приложения**
   ```bash
   flutter run
   ```
