## 📊 API Endpoints Checklist

### Аутентификация
- [ ] `POST /api/auth/login` - Вход в систему
- [ ] `POST /api/auth/register` - Регистрация
- [ ] `POST /api/auth/logout` - Выход
- [ ] `POST /api/auth/refresh` - Обновление токена
- [ ] `POST /api/auth/forgot-password` - Восстановление пароля
- [ ] `POST /api/auth/reset-password` - Сброс пароля

### Пользователи
- [ ] `GET /api/user/profile` - Получение профиля
- [ ] `PUT /api/user/profile` - Обновление профиля
- [ ] `POST /api/user/avatar` - Загрузка аватара
- [ ] `GET /api/user/preferences` - Настройки пользователя
- [ ] `PUT /api/user/preferences` - Обновление настроек

### Абонементы
- [ ] `GET /api/user/membership` - Текущий абонемент
- [ ] `GET /api/user/membership/history` - История абонементов
- [ ] `POST /api/memberships/purchase` - Покупка абонемента
- [ ] `POST /api/memberships/renew` - Продление абонемента
- [ ] `GET /api/memberships/plans` - Планы абонементов

### Тренировки
- [ ] `GET /api/classes/list` - Список занятий
- [ ] `GET /api/classes/{id}` - Детали занятия
- [ ] `POST /api/classes/book` - Запись на занятие
- [ ] `DELETE /api/classes/cancel/{id}` - Отмена записи
- [ ] `GET /api/user/bookings` - Мои записи
- [ ] `GET /api/events/list` - Список мероприятий
- [ ] `POST /api/events/register` - Регистрация на мероприятие

### Клубы и тренеры
- [ ] `GET /api/clubs/list` - Список клубов
- [ ] `GET /api/clubs/{id}` - Детали клуба
- [ ] `GET /api/trainers/list` - Список тренеров
- [ ] `GET /api/trainers/{id}` - Профиль тренера
- [ ] `POST /api/trainers/book` - Запись к тренеру

### Статистика
- [ ] `GET /api/user/stats/attendance` - Статистика посещений
- [ ] `GET /api/user/stats/duration` - Продолжительность тренировок
- [ ] `GET /api/user/stats/peak-hours` - Лучшее время посещения
- [ ] `GET /api/user/referral/xp` - Реферальные очки

### Уведомления
- [ ] `GET /api/user/notifications/settings` - Настройки уведомлений
- [ ] `PUT /api/user/notifications/settings` - Обновление настроек
- [ ] `GET /api/user/notifications/history` - История уведомлений
- [ ] `POST /api/notifications/send` - Отправка уведомления

### Акции
- [ ] `GET /api/promotions/list` - Список акций
- [ ] `GET /api/promotions/{id}` - Детали акции
- [ ] `POST /api/promotions/apply` - Применение промокода

### ИИ Ассистент
- [ ] `POST /api/ai/chat` - Отправка сообщения
- [ ] `GET /api/ai/history` - История чата
- [ ] `GET /api/ai/recommendations` - Персональные рекомендации
